
- 使用 **acme.sh**
- 指定 **Let's Encrypt** CA
- 使用 **Cloudflare DNS** 自动验证通配符域名
- 自动安装到 **Nginx**
- 自动续期并重载 Nginx
```
export CF_Token="epvpkWSiQerYNWdOBm-n7vh_vlnRAQA2bmluVWwK"
export CF_Account_ID="你的Cloudflare账号ID"
```

```bash
acme.sh --issue -d shensky.shop -d '*.shensky.shop' --dns dns_cf --server https://acme-v02.api.letsencrypt.org/directory

```
## 4️⃣ 安装证书到 Nginx 并自动续期

```bash
acme.sh --install-cert -d shensky.shop \
--key-file /etc/nginx/ssl/shensky.shop.key \
--fullchain-file /etc/nginx/ssl/shensky.shop.cer \
--reloadcmd "systemctl reload nginx"

```
